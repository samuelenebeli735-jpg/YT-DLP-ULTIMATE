# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project follows Semantic Versioning.

---

## [1.0.0] - 2026-07-02

### Added
- Initial release of the YT-DLP Ultimate Toolkit.
- Interactive command-line menu.
- Download best available video quality.
- Download videos in 720p MP4.
- Extract audio as MP3.
- List available video formats.
- Download playlists and channels.
- Configurable download folder.
- Optional `cookies.txt` support.
- Built-in yt-dlp update option.
- Automatic project folder creation.
- Initial project documentation.

### Planned
- Download history.
- Subtitle downloader.
- Thumbnail downloader.
- Custom quality selection.
- Download queue.
- Automatic browser detection.
- Better error handling.
- PowerShell edition.
