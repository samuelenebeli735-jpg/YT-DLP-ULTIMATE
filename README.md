# YT-DLP Ultimate

A professional Windows command-line toolkit built with Batch, yt-dlp and FFmpeg.

## Features

- Modular architecture
- Video downloads
- Audio extraction
- Playlist support
- Format selection
- Settings management
- Browser cookie support
- Download history
- Automatic updates

## Project Status

🚧 Currently under active development.

## Requirements

- Windows 10/11
- yt-dlp
- FFmpeg
